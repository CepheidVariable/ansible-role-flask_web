Flask Web Server
=========

This role installs a Flask web server on a CentOS 8 system.


Requirements
------------

None.


Dependencies
------------

Python must be installed.


Example Playbook
----------------

Example of implementation:

    - hosts: $SERVERS
      roles:
         - flask_web
